# SimpleTiktok
第五届字节跳动青训营项目

项目方案：https://bytedance.feishu.cn/docs/doccnKrCsU5Iac6eftnFBdsXTof#

实现了大部分的接口，只做了单机版本的

能跑通所有功能，接口测试完备

运行数据库:

`docker-compose up`

`redis-server`

运行程序：

`go run .`
